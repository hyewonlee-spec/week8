# Environment Audit Coach

A Vercel-ready React + Vite prototype for the Environment Audit Coach app.

## What is included
- Before Event flow
- After Event flow
- Calm older sibling coach tone
- AI-connected preview framing
- Structured summary cards

## Local run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Upload this folder to a new GitHub repository.
2. Import the repository into Vercel.
3. Vercel should detect **Vite** automatically.
4. Build command: `npm run build`
5. Output directory: `dist`

## Notes
This is currently a front-end prototype. It behaves like an AI coach, but it does not yet call a live AI backend.
